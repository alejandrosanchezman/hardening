Guia para control de BIOS

Para comenzar con esta guia deberemos iniciar ó reiniciar el pc
una vez estemos reiniciando para poder acceder de forma directa a la bios pulsaremos repetidamente f12.

la primera parte sobre la que nos vamos a centrar en la bios es la addicion de una contraseña de usuario.

Contraseña de usuario o power-On

Una vez dentro de la bios nos dirigimos al apartado de seguridad

una vez alli nos movemos hasta el apartado de Power-on password para añadir una contraseña para el usuario como se puede comprobar en la siguien imagen.

Contraseña de Administrador

Si continuamos en el mismo segmento podremos cambiar la contraseña del administrador.

Arranques externos

para poder realizar los cambios necesarios para que no nos puedan introducir un dispositivo externo a modo de arranque nos dirigiremos hasta la seccion de devices en la bios y buscaremos la seccion de network para deshabilitar el pxe.

Orden de arranque

el orden de arranque lo podremos modificar desde la seccion de startup una vez alli nos dirigiremos a primary boot sequence.

para poder ordenar los dispositivos que checkeara en busca de un sistema para arrancar.

  • normalmente los dispositivos estan ordenados de tal manera que primero comprueba el lector de disco y si no encuentra un elemento que cargar pasaria a la siguiente opcion la cual normalmente es el primer HDD donde se encontraria el sistema operativo.(este sistema virtual no permite enseñar los dispositivos de carga porque son inexistentes).

Otras opciones de seguridad

A continuacion vamos a añadir otras opciones de seguridad que aparecen en la bios del sistema.

  • contraseñas tanto de administrador como de usuario.

  • update del firmware de la bios para mantenerla siempre actualiza y evitar vulnerabilidades.

  • secure boot que nos permite evitar el funcionamiento de otro software no firmado durante el arranque.

  • device guard el cual añade restricciones para evitar problemas con malware en los dispositivos.
