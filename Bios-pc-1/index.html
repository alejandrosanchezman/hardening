Guia para control de BIOS

Para comenzar con esta guia deberemos iniciar ó reiniciar el pc una vez estemos reiniciando para poder acceder de forma directa a la bios pulsaremos repetidamente f12. el primer paso a realizar es añadir una contraseña de usuario:

Contraseña de usuario o power-On

Una vez dentro de la bios UEFI nos vamos al apartado de bios y bajamos hasta encontrar contraseña de usuario una vez alli nos dejara añadirla

Contraseña de Administrador

En la misma pestaña podremos encontrar justo encima del user password la password de administrador.

Arranques externos

En esta seccion podremos deshabilitar la opcion de arranque externo para evitar posibles arranques con intenciones maliciosas.

Otras opciones de seguridad

A continuacion vamos a añadir otras opciones de seguridad que aparecen en la bios del sistema.

  • Bootup numlock state se utiliza para comprobar el estado del numlock al arranque.
  • Security option te permite proteger la bios con una password para evitar el acceso sin autorizacion de usuarios.
